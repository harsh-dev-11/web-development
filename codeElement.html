HTML

Hyper Text Markup language is very easy to learn and used to structure the website.

HTML Boiler plate code

A boiler plate code is basically the essential and required code to run the html, without boiler plate code, output may not be as desired.

Here is the code:

           <!DOCTYPE html>
            <html lang="en" >

            <head>
                <meta charset="UTF-8" >
                <meta name="viewport" content="width=device-width, initial-scale=1.0" >
                <title>Code Element</title>
            </head >

            <body >

            </body >
            </html>
