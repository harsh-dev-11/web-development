HTML Ruby Element Examples

The <ruby> element represents ruby annotations, commonly used in East Asian typography to show pronunciation or meaning.

Basic Ruby Structure

漢字 ( かんじ )

Code:

<ruby>漢字 <rp>(</rp><rt>かんじ</rt><rp>)</rp></ruby>

Elements used:

  • <ruby> - Container for the ruby annotation
  • <rt> - Ruby text (pronunciation)
  • <rp> - Ruby parentheses (fallback for unsupported browsers)

Multiple Ruby Annotations

今日 ( きょう ) は 天気 ( てんき ) がいいです。

Code:

<ruby>今日 <rp>(</rp><rt>きょう</rt><rp>)</rp></ruby>は<ruby>天気 <rp>(</rp><rt>てんき</rt><rp>)</rp></ruby>がいいです。

Simple Ruby (without rp elements)

HTML HyperText Markup Language

Code: <ruby>HTML <rt>HyperText Markup Language</rt></ruby>

Note: Without <rp> elements, browsers that don't support ruby will show the annotation without parentheses.

Complex Ruby with Multiple Parts

攻 殻 機 動 隊 ( こう かく き どう たい )

Code:

<ruby>
  <rb>攻</rb><rb>殻</rb><rb>機</rb><rb>動</rb><rb>隊</rb>
  <rp>(</rp><rt>こう</rt><rt>かく</rt><rt>き</rt><rt>どう</rt><rt>たい</rt><rp>)</rp>
</ruby>

Additional element:

  • <rb> - Ruby base (the characters being annotated)

Browser Support Note

The <rp> elements provide fallback parentheses for browsers that don't support ruby annotations. In browsers that do support ruby, the <rp> content is hidden and the <rt> content appears above the base text.

In browsers without ruby support, you'll see: 漢字(かんじ)

In browsers with ruby support, you'll see the pronunciation above the characters.
