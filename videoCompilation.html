Foundational Skills for Web Development

Web development is vast process. There are many technologies that are required in order to become the better developer. Although, there are some foundational skills that are required to become a good web developer. In this video compilation, we will explore some of these foundational skills.

HTML : Structure of website

HTML stands for 'Hyper Text Markup Language'. It is the basic structure of any website. It inlcude the tags/elements to create the atomic structure of a website. It doesn't include any designing, just defines the structure of a website

CSS : designing the webpage

CSS stands for: Cascading Style sheet, it includes the all stuff in order to desing the webpage and make it more accessible and user friendy. CSS is a vast technology, it does have other library and frameworks and with the help of those frameworks, we will be able to make interactive webpages.

Javascript : Logic of website

JavaScript defines the behavious and logic behind the various elements of a website. It is the programming language and have frameworks like : Next.js, React and Angular. We can do lot of other things with the help of javascript besides the web development, so it is really worthfull to invest learning the javascript
